## 🚀 立即使用

### 1. 配置API密钥

```bash
cd /Users/a1234/.hermes/skills/devops/changshu-dev-assistant

# 编辑配置文件
vi config.yaml
```

在 `ai` 部分添加：

```yaml
ai:
  provider: ""  # 通过环境变量LLM_PROVIDER设置
  api_key: ""  # 通过环境变量LLM_API_KEY设置
  model_name: ""  # 通过环境变量LLM_MODEL_NAME设置
  temperature: 0.7
  max_tokens: 2000
```

### 2. 启动对话

```bash
# 启动交互式对话
./start.sh chat

# 或使用快捷命令
csa chat
```


### 配置API后

```
昌叔助手 > 如何优化达梦数据库查询？

要优化达梦数据库查询，可以从以下几个方面入手：

1. **索引优化**
   - 为经常查询的字段创建合适的索引
   - 避免在索引列上使用函数
   - 定期维护索引统计信息

2. **SQL语句优化**
   - 避免使用SELECT *，只查询需要的字段
   - 合理使用JOIN，避免不必要的关联
   - 使用EXISTS替代IN子查询

3. **查询计划分析**
   - 使用EXPLAIN查看执行计划
   - 识别全表扫描和低效操作
   - 根据执行计划调整SQL

4. **数据库配置**
   - 调整缓冲区大小
   - 优化排序和哈希操作
   - 合理设置并行度

需要我详细说明某个方面吗？
```
